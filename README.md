# Macroweaver: AI Economic Policy Advisor

Macroweaver is an interactive, web-based simulator designed to model the macroeconomic impacts of government policy changes. By adjusting key economic levers, users can observe real-time projections on national output, inflation, and employment, accompanied by plain-language analytical explanations generated by artificial intelligence.

## Live Demo

**[Visit the Live Website Here](https://macroweaver-sigma.vercel.app/)** *(Note: Replace this URL with your actual Vercel or custom domain link once deployed)*

## Project Overview

The application utilizes a simplified Keynesian macroeconomic framework integrated with an AI explanation layer. It serves as both an educational tool for understanding macroeconomic principles and a demonstration of full-stack engineering integrating data modeling with Large Language Models (LLMs).

### Key Capabilities

*   **Interactive Simulation:** Users can adjust five primary policy levers: Corporate and Income Tax Rates, Agricultural Subsidies, Central Bank Interest Rates, Government Spending, and Import Tariffs.
*   **Real-time Modeling:** Applying established sensitivity coefficients, diminishing return parameters, and fiscal multipliers to calculate projected deviations from baseline indicators.
*   **Comparative Analysis:** Side-by-side scenario modeling allowing users to evaluate divergent policy approaches (e.g., Expansionary vs. Conservative).
*   **AI-Driven Context:** Integration with the Groq API to formulate clear, non-technical macroeconomic analyses of the simulated outcomes.
*   **Data Foundation:** Baseline metrics are derived from historical World Bank Development Indicators for India.

## System Architecture

The project is structured as a decoupled application:

*   **Frontend Client:** A Single Page Application (SPA) built with React and Vite. It utilizes a custom design system inspired by modern editorial interfaces, featuring responsive data visualization via Recharts.
*   **Backend Application Programming Interface (API):** A high-performance Python service built on FastAPI. It manages the economic modeling logic, data ingestion, and asynchronous communication with external AI providers.

## Tech Stack

*   **Frontend Environment:** React, Vite, JavaScript (ES6+), Vanilla CSS
*   **Data Visualization:** Recharts
*   **Backend Infrastructure:** Python 3, FastAPI, Uvicorn, Pydantic
*   **Data Processing:** Pandas, NumPy
*   **Artificial Intelligence:** Groq API (Llama 3.3 Versatile)
*   **External Data Integration:** World Bank API (`wbgapi`)

## Local Development Setup

To run Macroweaver locally, both the backend API and the frontend client must be initialized.

### Prerequisites

*   Python 3.9 or higher
*   Node.js 18.0 or higher
*   A valid Groq API Key (available at console.groq.com)

### 1. Backend Initialization

Navigate to the project directory and execute the following commands to establish the Python environment:

```bash
cd backend
python -m venv venv

# Activate the virtual environment
# Windows:
venv\Scripts\activate
# macOS/Linux:
# source venv/bin/activate

pip install -r requirements.txt
```

Configure your environment variables:
1. Copy `.env.example` located in the root directory to `.env`.
2. Insert your Groq API key into the `GROQ_API_KEY` field.

Start the FastAPI server:

```bash
uvicorn backend.main:app --reload
```
The API will be available at `http://localhost:8000`. Interactive API documentation is accessible at `http://localhost:8000/docs`.

### 2. Frontend Initialization

In a separate terminal session, navigate to the frontend directory:

```bash
cd frontend
npm install
npm run dev
```
The user interface will be available at `http://localhost:5173`.

## Deployment Guidelines

Macroweaver is configured for deployment across standard Platform-as-a-Service (PaaS) providers. Detailed step-by-step instructions are available in the `DEPLOYMENT.md` file located in the repository root.

*   **Backend:** Recommended deployment via Render or Railway using the included `Procfile` and `requirements.txt`.
*   **Frontend:** Recommended deployment via Vercel or Netlify. Ensure the `VITE_API_BASE_URL` environment variable is set to your production backend URL.

## Disclaimer

This software is provided as an educational reference tool. The underlying mathematical model relies on simplified macroeconomic assumptions that do not capture the profound complexity, nonlinear feedback loops, and global interdependencies of real-world economies. It should not be utilized for formal policy formulation or financial advisory purposes.
